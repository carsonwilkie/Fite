## Screenshot Guide

Apple requires screenshots for two device sizes minimum. App Store Connect calls them:
- **6.9" Display** (iPhone 16 Pro Max) — 1320 × 2868 px
- **6.5" Display** (iPhone 14 Plus / 15 Plus) — 1284 × 2778 px — *also satisfies 5.5" requirement*

You can capture these using the iOS Simulator in Xcode. Set the simulator device to **iPhone 16 Pro Max**, run your app, and press **Cmd+S** to save a screenshot.

---

### Tips for Clean Screenshots
- Use a real device or the highest-resolution simulator
- Make sure the status bar shows a clean time (e.g. 9:41 AM) — in Simulator, use **Cmd+Shift+H** to trigger the clean status bar
- Use dark mode (the app is dark-only, so this is automatic)
- Avoid showing personal email or real user data in any screen
