How to install

Gmail

Open this file in a browser, select the rendered preview with your cursor, copy it, then paste into Settings → See all settings → General → Signature. Set it as the default for new mail and replies.

Outlook (web)

Select + copy the rendered preview, go to Settings → Mail → Compose and reply, paste into the signature editor, name it "Fite Support," and apply to new + reply messages.

Apple Mail

Mail → Settings → Signatures. Create a new signature, uncheck "Always match my default message font," then paste the preview. Assign it to the support account.

Logo note: the img src must point to a publicly accessible URL. https://fitefinance.com/Fite_Premium_NB.png is pre-wired since that asset already ships in /public. Swap it for a dedicated wordmark if you want a cleaner mark.
